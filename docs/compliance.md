# Compliance Notes

This product is positioned as statistical recommendation software only.

- It must not sell lottery tickets, broker purchases, issue tickets, collect betting stakes, or process betting payments.
- Number generation, model scores, and backtests are educational/statistical signals, not promises of winning.
- Users should verify rules, draw results, and prize information through official public sources.
- The client and server should show age and responsible-purchase reminders before premium features.
- Membership payments are for software features and reports only; they are not lottery stakes.

- Double Color Ball rules: https://www.cwl.gov.cn/c/2018-10-12/417937.shtml
- Super Lotto rules: https://m.lottery.gov.cn/ksjz/m/yxgz_dlt/
